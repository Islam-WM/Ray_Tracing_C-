Detailed Description

Virtual Object class that acts as the base for all movable objects in the scene.

Objects are all non-temporary elements that have a location in the scene.(primitive Objects, camera, lights, ...)

◆ ref_index

double Object::ref_index
protected

ref_index stores the refraction index of the object

The refraction index is used to calculate the refraction angle when going from one object/medium into another.

Note
Not every child object needs a refraction index, e.g. Camera. For these objects ref_index is set to a default value
