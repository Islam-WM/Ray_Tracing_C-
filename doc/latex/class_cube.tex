\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_cube}{Cube}} Class is a child class of \mbox{\hyperlink{class_primitive}{Primitive}}. 

The \mbox{\hyperlink{class_cube}{Cube}} Class provides a composition of six planes that can be rendered.
