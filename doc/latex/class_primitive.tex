\doxysubsection{Detailed Description}
Virtual \mbox{\hyperlink{class_primitive}{Primitive}} class that acts as the base for all renderable objects in the scene. 

The \mbox{\hyperlink{class_primitive}{Primitive}} Class builds up on the \mbox{\hyperlink{class_scene___element}{Scene\+\_\+\+Element}} Class and adds the functionality needed for an object to be rendered in the scene. \begin{DoxySeeAlso}{See also}
\mbox{\hyperlink{class_scene___element}{Scene\+\_\+\+Element}} While \mbox{\hyperlink{class_scene___element}{Scene\+\_\+\+Element}} is the base for all non-\/temporary Elements in the scene, the \mbox{\hyperlink{class_primitive}{Primitive}} Class is not a parent of for example the camera that does not need to be included in the rendered image.
\end{DoxySeeAlso}
Therefore the \mbox{\hyperlink{class_primitive}{Primitive}} Class provides methods to compute e.\+g. reflection\+\_\+angles and color values.
