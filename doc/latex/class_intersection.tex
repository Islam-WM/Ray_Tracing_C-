\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_intersection}{Intersection}} Class is a child class of \mbox{\hyperlink{class_primitive}{Primitive}}. 

The \mbox{\hyperlink{class_intersection}{Intersection}} Class provides a composition of several primitives that can be rendered.
