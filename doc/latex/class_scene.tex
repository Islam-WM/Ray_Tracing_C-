\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_scene}{Scene}} class stores the objects and lights of a \mbox{\hyperlink{class_scene}{Scene}} and takes care of the setup of the render. 

The \mbox{\hyperlink{class_scene}{Scene}} class provides functions to read the settings and initiates the render process. It stores the objects, lights and camera as members.
