\mbox{\hyperlink{class_ray_a9d678eca25578b3db95cd4a4e085b45d}{trace\+\_\+ray()}} traces a \mbox{\hyperlink{class_ray}{Ray}} through the scene 

\mbox{\hyperlink{class_ray_a9d678eca25578b3db95cd4a4e085b45d}{trace\+\_\+ray()}} is the function that checks if a ray hits an object. If it does it computes a return\+\_\+color for the ray which consists of ambient light, direct light through light sources, reflection, refraction,... Inorder to compute reflection, refraction,... new rays are send recursively.
