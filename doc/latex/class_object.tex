\doxysubsection{Detailed Description}
Virtual \mbox{\hyperlink{class_object}{Object}} class that acts as the base for all movable objects in the scene. 

Objects are all non-\/temporary elements that have a location in the scene.(primitive Objects, camera, lights, ...)
