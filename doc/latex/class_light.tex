\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_light}{Light}} class implements a simple point light. 

\mbox{\hyperlink{class_light}{Light}} has a location and a color.

\begin{DoxyNote}{Note}
The \mbox{\hyperlink{class_light}{Light}} class has been implemented as a child of \mbox{\hyperlink{class_scene___element}{Scene\+\_\+\+Element}} even though it only uses the location, incase one wants to later create different light sources and not just point lights. 
\end{DoxyNote}
